# student_data_analysis
# 📊 Student Performance Analysis Using Python & Pandas

## 📌 Project Overview

This project performs Exploratory Data Analysis (EDA) on a student dataset using Python and Pandas. The objective is to analyze student academic performance, attendance records, grades, and overall statistics to gain meaningful insights.

The project demonstrates fundamental data analysis operations such as data cleaning, aggregation, filtering, grading, and performance evaluation.

---

## 🎯 Objectives

* Analyze student academic records.
* Calculate total marks and percentages.
* Assign grades based on performance.
* Identify top and lowest-performing students.
* Analyze attendance records.
* Compare performance across genders.
* Export processed data to a CSV file.

## 🔍 Analysis Performed

### 1. Data Loading

* Loaded dataset using Pandas.
* Displayed initial records.

### 2. Data Inspection

* Checked dataset information.
* Verified data types.
* Counted rows and columns.

### 3. Data Cleaning

* Checked for missing values.
* Checked duplicate records.

### 4. Academic Performance Analysis

* Calculated Total Marks.
* Calculated Percentage.
* Assigned Grades.

### Grade Criteria

| Percentage | Grade |
| ---------- | ----- |
| 90+        | A+    |
| 80–89      | A     |
| 70–79      | B     |
| 60–69      | C     |
| Below 60   | D     |

### 5. Performance Insights

* Top-performing student
* Lowest-performing student
* Average marks per subject
* Students scoring below average
* Highest attendance holder
* Gender-wise average performance
